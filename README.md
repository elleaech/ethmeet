## Installation
```shell script
git clone https://github.com/mstr-Wolf/browserBot.git
```

### Dependancies
#### Package Installer
```shell script
pip3 install -r requirements.txt
```
#### Pypy Compiler
```shell script
pypy3 -m ensurepip
pypy3 -m pip install -r requirements.txt
```
